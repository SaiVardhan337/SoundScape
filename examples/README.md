# SoundScape API Examples 💡

This directory contains standalone usage examples showing how to interact with the SoundScape backend programmatically.

## Running the Examples

1. Start the SoundScape engine server:
   ```bash
   python main.py
   ```

2. In a separate terminal window, run the quickstart script:
   ```bash
   python examples/quickstart.py
   ```

## Included Scripts

- **`quickstart.py`**: Interacts with REST endpoints to fetch & update notes, record focus sessions, and query weekly focus statistics using Python standard library (`urllib`).
